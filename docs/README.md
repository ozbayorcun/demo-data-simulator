# demo-data-simulator docs

This folder mixes public maintainer docs, release notes, and historical planning
records. The README remains the primary user guide.

## Public Maintainer Docs

- `release-runbook.md` - npm release process and trusted-publishing setup notes.
- `pre-publish-proof-bundle.md` - local proof bundle for public package readiness.
- `scenario-packs-proof-reports-plan.md` - next feature plan for scenario packs and proof reports.

## Product Context

- `PRD.md` - product requirements and positioning context.
- `REAL-REPO-EVAL.md` - maintainer-owned validation notes from real repos.
- `REVIEW-CONSENSUS.md` - early product/architecture review synthesis.
- `CONSENSUS-REVIEW-2026-06-10.md` - pre-public review and fixes.

## Examples

- `../examples/field-service/README.md` - tiny app-shaped fixture.
- `../examples/field-service/dashboard/README.md` - local static dashboard proof.

## Notes

Generated demo data is synthetic. This project does not anonymize production
records and should not be used as a production data masking tool.
